# История изменений DevEvents

## [1.0.0] - 2025-01-07 (Initial)

### Первый релиз 🎉

#### ✨ Основные возможности

**Лента событий**

- Агрегация IT-мероприятий из публичных Telegram-каналов
- Pull-to-refresh для обновления списка
- Автоматическое фоновое обновление каждые 6 часов
- Красивые карточки событий с изображениями

**Фильтры и сортировка**

- Фильтрация по категориям: Android Dev, DevOps, Game Dev, Other
- Быстрые фильтры: «Сегодня», «На этой неделе», «Онлайн», «Бесплатные»
- Сортировка по дате или названию
- Сохранение состояния фильтров между сессиями

**Детальная информация**

- Полное описание события с сохранением форматирования
- Отображение даты, времени, места проведения
- Информация об организаторе, формате, цене
- Теги и категории
- Дополнительная информация: призовой фонд, дедлайны регистрации

**Избранное**

- Сохранение интересных событий
- Автоматические напоминания за 24 часа и 1 час до начала
- Экспорт избранного в JSON
- Импорт избранного из JSON с валидацией
- Восстановление напоминаний при импорте

**Экспорт и интеграции**

- Поделиться событием (текст с ссылкой)
- Экспорт события в формат iCalendar (.ics)
- Добавление в системный календарь
- Deep links: `eventparser://event/<eventId>`

**Уведомления**

- Напоминания о событиях из избранного
- Уведомления о новых найденных событиях
- Умное планирование уведомлений
- Открытие события при нажатии на уведомление

**Интерфейс**

- Material Design 3
- Темная и светлая тема
- Автоматическая тема (следует за системой)
- Плавные анимации и переходы
- Адаптивный Bottom Bar
- Splash Screen
- Онбординг при первом запуске

**Настройки**

- Выбор темы оформления
- Быстрый доступ к системным настройкам уведомлений
- Очистка кэша UI-настроек
- Поддержка разработчика
- Обратная связь
- Информация о версии

#### 🏗️ Архитектура

- Clean Architecture (Domain, Data, Presentation)
- MVI паттерн с Orbit MVI
- Dependency Injection с Koin
- Room для локального хранения
- DataStore Preferences для настроек
- WorkManager для фоновых задач
- Coroutines и Flow для асинхронности

#### 🛡️ Безопасность и стабильность

- Обработка крэшей с сохранением логов
- Exponential backoff для retry-логики
- Дедупликация событий на уровне парсера и БД
- Автоматическая очистка прошедших событий
- Offline-first подход с кэшированием

#### 🔧 Технические особенности

- Парсинг Telegram-каналов через Jsoup
- Умное извлечение дат с нормализацией
- Поддержка дайджестов (один пост → несколько событий)
- Эвристический анализ текста для определения типа события
- Сохранение форматирования текста (переносы строк, ссылки)

#### 📦 Оптимизация

- ProGuard для минификации кода
- Resource shrinking
- WebP формат для изображений
- Эффективное использование памяти
- Минимальные разрешения

#### 🌍 Локализация

- Полная поддержка русского языка
- Все строки в strings.xml
